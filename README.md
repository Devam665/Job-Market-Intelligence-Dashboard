# 📊 Job Market Intelligence Dashboard

## 🚀 Project Overview

The Job Market Intelligence Dashboard is an end-to-end Data Analytics and Business Intelligence project designed to analyze hiring trends in the data industry. The project combines job postings from multiple sources, including Naukri, Glassdoor, and other job datasets, to provide insights into the current demand for Data Analysts, Data Scientists, and related roles.

Using Python for data preprocessing and Power BI for visualization, this project analyzes over **17,000+ job postings** to identify hiring patterns, skill demand, experience requirements, and market opportunities.

---

## 🎯 Objectives

The project aims to answer the following business questions:

* Which technical skills are most in demand?
* What are the key differences between Data Analyst and Data Scientist roles?
* Which companies are hiring the most?
* What experience levels are employers seeking?
* Which technologies provide the greatest career advantage?
* How can job seekers align their skills with market demand?

---

## 📂 Dataset Information

The project integrates multiple datasets collected from different job platforms:

* Naukri Job Dataset
* Glassdoor Data Analyst Dataset
* Clean Jobs Dataset
* Data Science Jobs Dataset

After cleaning and merging, a unified dataset containing **17,000+ job postings** was created.

**Dataset Included:**
The complete dataset is provided in compressed ZIP format within this repository to comply with GitHub file size limitations.
